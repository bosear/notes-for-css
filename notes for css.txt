Background фон картинку на всю страницу:
////////////////////////////////////////
высота в браузере:
viewport->html->body
по умолчанию всегда авто - сжимается под контент, нет контента - нет высоты
чтобы отобразить элеммент на всю страницу необходимо <html> задать высоту 100% (будет в таком случае брать от экрана - viewport-а)
background-cover расстягивает, но вроде как сохраняет пропорции
Два решения:
1) 
	html, body {
		height: 100%;
	}
	.wrapper {
		background: url('background.jpg') no-repeat;
		background-size: 100% 100%; // or cover
		height: 100%;
	}
2)  body { // or html
		background: url(new-york-background.jpg) no-repeat center center fixed;
		background-size: cover;
	}
	Для wrapper-а уже не сработает(
3)
	Поставить элементу - wrapper-у height: 100vh;
